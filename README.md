# yaps-releases

Public release assets for Yaps.

## Android Keyboard Language Packs

The Android keyboard downloads optional language packs from:

```text
https://raw.githubusercontent.com/richawo/yaps-releases/main/android-keyboard-language-packs
```

The directory preserves the runtime-relative paths expected by the app, for
example:

```text
android-keyboard-language-packs/keyboard/nlp/de-DE/words.tsv
```

`android-keyboard-language-packs/manifest.json` records the published file list,
sizes, and SHA-256 hashes for quick audits.
